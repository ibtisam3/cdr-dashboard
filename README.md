# CDR Dashboard

Hi, I’m **IBTISAM SHEIK-ALI**, and this is my Call Data Records (CDR) Dashboard project! 🎉  

This dashboard is a modern, interactive way to explore call data. I built it with **React**, **Vite**, **Tailwind CSS**, and used **shadcn/ui** for the UI components. The dashboard includes KPI cards, charts, and a table of recent call logs — all responsive and easy to use.  

---

## Live Demo

Check it out live here: [CDR Dashboard](https://cdr-dashboard.vercel.app/)

---

## Features

- **KPI Cards** – Quickly see total calls, total cost, average duration, successful and failed calls.  
- **Interactive Charts** – Visualize call duration, cost, activity over time, and distribution by city.  
- **Recent Call Logs** – Paginated table showing the latest calls with filtering options.  
- **Responsive Design** – Works seamlessly on desktop and mobile devices.  
- **Shadcn/UI Integration** – Uses cards and tables from shadcn for a clean and consistent design.
